A WordPress plugin to use API Direct Delivery API for getting quote from Uber and displaying it on checkout page. 
  
== Description ==
  
This plugin is lite version of our plugin that can help integrate Uber API to setup delivery from a pickup address to given dropoff address. 
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.
3. Go to Woocommerce > Settings > Shipping > Uber Direct Delivery option and provide Customer ID, Client ID, Client Secret and save.
4. On the same page press get OAuth Token and save the page again. Remember you need to get new OAuth Token every 30 days.
